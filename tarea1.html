Cómo hacer tu propia página web 101

El primer paso es descargar la aplicación Visual Studio Code. Una vez abriendo la aplicación, estás listo para comenzar tu primer código. Para no comenzar de cero con el sitio Web, entraremos a la página “html5up.net”, en el cual encontraremos diversas plantillas con formatos originales de un sitio Web. Al escoger uno, todo lo que debes hacer es descargarlo. Una carpeta completa se descargará en tu computadora, conteniendo los archivos y los códigos de todo dentro de tu plantilla para que puedas editarla y trabajar sobre ella.

Algunos de estos son: assets, elements.html, generic.html, images, index.html, license.txt. Desde la aplicación de Visual Studio Code, abriremos esa carpeta, en donde podremos apreciar todos los códigos de cada elemento. Una vez hecho eso, podemos elegir crear nuevas pestañas y poner enlaces que nos redirijan a ellas; cambiar las imágenes, títulos, subtítulos y básicamente toda la información que queramos. A la par en que hacemos esto, damos “cmd+S” para guardar todos los cambios que hacemos y vamos a la página web en construcción, le damos refresh, y vemos si los cambios se ven como nosotros queríamos.
